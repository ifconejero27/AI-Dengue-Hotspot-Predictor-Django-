# Predicting Dengue Hotspots in Malabon City through AI
An AI-powered web dashboard built using Python and Django to predict dengue outbreaks in Malabon City based on weather-case analysis.
**Capstone Project | BS Information Technology**

## Project Overview
A monolithic web application engineered using the Python-powered **Django framework**. The system processes historical weather metrics and localized dengue case statistics to serve AI-driven predictive insights and visualize high-risk community outbreak clusters in Malabon City.

## Tech Stack & Architecture
* **Backend Framework:** Python / Django (MVC Architecture)
* **Database:** SQLite / MySQL
* **Data Visualization:** Geospatial Mapping Scripts & Integrated Chart Libraries
* **Version Control:** Git & GitHub

## Key Features Implemented
* **Server-Side Routing:** Robust data processing and backend routing structures handled natively via Django.
* **Geospatial Mapping:** Dashboard layout mapping coordinates to pinpoint outbreak severity clusters visually.
* **Weather-Case Correlation:** Backend data pipelines syncing weather pattern inputs directly against local metrics.
